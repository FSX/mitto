Mitto
=====

Mitto is a Thrift_ implementation for Tornado_. Development has just started
and there is no usable code right now. Mitto wil include a Thrift client to
communicate with Thrift servers and a code generate to generator code from the
Thrift IDL.

Mitto is the Latin word for "`to send`_".

.. _Thrift: http://thrift.apache.org/
.. _Tornado: http://www.tornadoweb.org/
.. _to send: http://en.wiktionary.org/wiki/mitto
